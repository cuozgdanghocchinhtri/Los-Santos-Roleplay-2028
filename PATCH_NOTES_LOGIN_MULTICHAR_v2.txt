LS:RP Login + Multi-Character v2
================================

Built against the exact LSRP-CURRENT.zip source supplied on 2026-07-23.

Flow:
1. Connect -> enter OOC account username.
2. Existing account -> BCrypt password login.
3. New account -> BCrypt registration.
4. Load 3 character slots.
5. Empty slot -> create First_Last character.
6. Existing slot -> load character data from MySQL.
7. Set in-game name to character name and spawn at Ganton.
8. Character state is saved on disconnect.

Database:
Import database/migrations/001_account_characters.sql once.
The existing database_structure.sql is intentionally not modified.
